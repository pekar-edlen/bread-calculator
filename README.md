# 🍞 Калькулятор домашнего хлеба

Интерактивный калькулятор рецептов для домашней выпечки хлеба.

**Открыть:** [pekar-edlen.github.io/bread-calculator](https://pekar-edlen.github.io/bread-calculator/)

## Возможности

- Расчёт рецепта по составу муки и % белка
- Поддержка смеси из нескольких видов муки
- Режимы: дрожжи / закваска
- Холодная ферментация (overnight)
- Расчёт температуры воды по методу Кальвеля
- Пресеты: белый, pain de campagne, ржаной, спельта, бриошь, халла
- График с планировщиком времени и интерактивным чек-листом
- Умный расчёт температуры духовки по составу теста

## Оборудование

- KitchenAid Artisan
- Духовка Siemens электрическая до 300°C с конвекцией
- Чугунный dutch oven

## Как обновить сайт

1. Получить новый `bread-calculator-dist.zip`
2. Распаковать → открыть папку `dist`
3. В репозитории: **Add file** → **Upload files**
4. Перетащить все файлы из `dist`
5. **Commit changes** — сайт обновится через 1–2 минуты
